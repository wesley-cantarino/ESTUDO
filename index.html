What's..:

  • HTML (Hypertext Markup Language)
    - É o coração de tudo. HTML é uma linguagem de marcação (não confundir com linguagem de programação). É fácil de aprender e bem simples.
  • CSS (Cascading Style Sheets)
    - É oque da "vida" em cores ao html. Requer mais paciência para aprender pois são muitos comandos e as vezes "ou quase sempre" as coisas sai um pouco (ou muito) do controle.
  • SASS (Syntactically awesome style sheets)
    - É o css com superpoderes. ao se implementar no site, o nível de díficuldade aumenta.
  • JavaScript
    - Não é java. É oque da vida ao site e js pode ser usado para muito mais coisas <3;
  • Node js
    - Permite rodar js fora do navegador.
  • JQuery
    - JQuery != js. A ideia do Jquery é escrever pouco e fazer muito. Ou seja, com poucas linhas de um resultado alto;
  • PHP
    Hypertext Preprocessor. big <3
  • SQL
    Structured Query Language. É usada para banco de dados. melhor exemplo é mySQL.
  • XML
    eXtensible Markup Language. É mais fácil e simples que html. Serve para compartilhar informações entre programas

1º forma é montar normalmente usando html, css e javascript e depois:
    Usar Electron para converter para desktop.

Electron

Electron ou atom shell permite criar aplicativos de desktop usando html, css e js. Tudo isso rodando no node.js e Chromium.

Desvantagem

  • Tamanho final >= 145 mB mas se colocar .zip fica próximo de 55 mB.
  • Outras coisas como porta serial não funciona com js.

Ionic

Ionic permite criar aplicativos mobile usando html, css e js.
